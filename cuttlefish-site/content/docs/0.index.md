## Key Concepts

### Multi-Agent System
Cuttlefish uses specialized agents (Orchestrator, Coder, Critic) that work together to solve problems. Each agent is optimized for its role through category-based model routing.

### Category-Based Model Routing
Different tasks are routed to different AI models based on their category:
- **deep**: Complex reasoning tasks (Claude Sonnet 4.6)
- **quick**: Fast, simple tasks (Claude Haiku 4.5)
- **ultrabrain**: Strategic planning (Claude Opus 4.6)
- **unspecified-high**: General high-effort tasks

### Docker Sandboxes
Each project runs in an isolated Docker container with configurable resource limits. This ensures projects don't interfere with each other.

### Templates
Start new projects from proven templates. Templates define project structure, dependencies, and initial configuration.
